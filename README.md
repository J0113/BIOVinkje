# BIOVinkje®

Compliance zonder daadkracht.

Genereer per BIO2-maatregel een professioneel klinkend excuus waarom uw organisatie er nét niet aan voldoet. Bestuurlijk verantwoord. Auditgereed. Vrijblijvend.

> *"Echte beveiliging kost tijd, geld en daadkracht. Een goed geformuleerd excuus kost één vergadering."*

## Demo

[BIOVinkje.jolle.dev](https://BIOVinkje.jolle.dev/)

## Wat het doet

1. Kies een BIO2-maatregel (zoek op nummer of trefwoord, bv. `8.16` of `logging`)
2. Klik **Genereer nieuw excuus** — krijg een bestuurlijke duiding
3. Niet tevreden? **Genereer beter excuus** escaleert de toon
4. Kopieer naar auditdossier met één klik
